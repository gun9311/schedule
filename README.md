# Fit Style
## Creating a web application for a Fitness club  

## How to install the project
[Guide](https://drive.google.com/file/d/1fZQNrWqWu4uRt_kzoFVKoUoaoMF8iZJJ/view?usp=sharing "Guide =)")

## Technology Stack
##### FrontEnd:
- HTML + CSS + JS
- Node.js (for React.js)
- React.js 
- Bootstrap 4
- Axios
##### BackEnd:
- Java
- Spring Data JPA
- Spring MVC
- PostgreSQL
