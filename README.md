# Fit Style
## Creating a web application for a Fitness club  
## Technology Stack
##### FrontEnd:
- HTML + CSS + JS
- Node.js (Для React.js)
- React.js 
- Bootstrap 4
- Axios
##### BackEnd:
- Java
- Spring Data JPA
- Spring MVC
- PostgreSQL

#### FrontEnd

Download Node.js: [Скачать](https://nodejs.org/en/ "Node.js =)")

npm i
